php.tools

PHP coding style formatter - Tooling for formatting, testing and code coverage

View project on GitHub

Installation

php composer.phar require dericofilho/fmt

What does the Code Formatter do?

vendor/bin/fmt.php filename filename2 ... filenameN

K&R configuration

Before After 
<?php
for($i = 0; $i < 10; $i++)
{
if($i%2==0)
echo "Flipflop";
}

<?php
for ($i = 0; $i < 10; $i++) {
    if ($i%2 == 0) {
        echo "Flipflop";
    }
}

<?php
$a = 10;
$otherVar = 20;
$third = 30;

<?php
$a        = 10;
$otherVar = 20;
$third    = 30;

echo "Flipflop";
}

        echo "Flipflop";
    }
}
Note the identation of 4 spaces. 
<?php
class A {
function a(){
return 10;
}
}

<?php
class A
{
    public function a()
    {
        return 10;
    }
}
Note the braces position, and the visibility adjustment in the method a().
